# Python Exercises

This is a list of my Python exercises.

## Band Name Generator

This script generates a band name based on the city you grew up in and the name of your pet.

## Tip Calculator

Based on the bill amount and different tip percentages, divided by the number of people, this tip calculator will show the amount each person should pay.

> Format the result to 2 decimal places.

## Rock Paper Scissors

A rock, paper, scissors game. Rock beats scissors, scissors beat paper, and paper beats rock.

## Password Generator

This script generates a password based on the number of letters, symbols, and numbers entered by the user.

## Hangman

Hangman is a guessing game. Player tries to guess it by suggesting letters within a certain number of guesses.
